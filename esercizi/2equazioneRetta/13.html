Esercizio 1


Risolvere il seguente sistema utilizzando il metodo di riduzione: \[ \begin{cases} 3x - y + 8 = 0 \\\\ 2x + 4y + 1 = 0 \end{cases} \]
Soluzione: \[ \begin{cases} x = -\dfrac{33}{14} \\\\ y = \dfrac{13}{14} \end{cases} \]




Esercizio 2


Individuare le coordinate del punto \(P\) di intersezione tra le rette \(r\) ed \(s\) di equazioni \[ r:\quad 3x + 2y - 3 = 0 \] \[ s:\quad 4x -2y + 1 = 0 \]
Soluzione: \(\,\,\, P\left(\dfrac{2}{7}\,;\,\,\dfrac{15}{14}\right) \)




Esercizio 3


Individuare le coordinate del punto \(Q\) di intersezione tra le rette \(r\) ed \(s\) di equazioni \[ r:\quad 5x + 2y - 1 = 0 \] \[ s:\quad -10x - 4y + 2 = 0 \]
Soluzione:
Le rette \(r\) ed \(s\) sono coincidenti, quindi il sistema ha infiniti punti di intersezione.




Esercizio 4


Individuare il valore del parametro \(k\) tale che la retta di equazione \[ r:\quad 5kx + (2k + 1)y - 1 = 0 \] non abbia intersezioni con la retta di equazione \[ s:\quad kx - y + 1= 0 \]
Soluzione: \(\,\,\, k = -3 \,\,\,\) oppure \(\,\,\,k = 0\)
