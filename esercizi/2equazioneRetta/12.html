Esercizio 1


Risolvere il seguente sistema utilizzando il metodo di sostituzione: \[ \begin{cases} 3x - y = 2 \\\\ 2x -y - 1 = 0 \end{cases} \]
Soluzione: \[ \begin{cases} x = 1 \\\\ y = 1 \end{cases} \]




Esercizio 2


Risolvere il seguente sistema utilizzando il metodo di riduzione: \[ \begin{cases} 4x - 3y + 1 = 0 \\\\ 2x + 5y + 7 = 0 \end{cases} \]
Soluzione: \[ \begin{cases} x = -\dfrac{10}{13} \\\\ y = -\dfrac{9}{13} \end{cases} \]




Esercizio 3


Individuare le coordinate del punto \(P\) di intersezione tra le rette \(r\) ed \(s\) di equazioni \[ r:\quad 2x + 9y - 8 = 0 \] \[ s:\quad -x -3y +1 = 0 \]
Suggerimento:
Il nostro obiettivo è individuare le coordinate del punto dove le rette \(r\) ed \(s\) si intersecano, ovvero si incontrano. Diamo nome a queste misteriose coordinate:

  • chiamiamo \(a\) la coordinata \(x\) del punto \(P\);

  • chiamiamo \(b\) la coordinata \(y\) del punto \(P\);

Poiché è intersezione delle due rette, in particolare il punto \(P\) appartiene alla retta \(\boldsymbol{r}\). Ovvero le coordinate \((a\,;\,\,b)\) di \(P\) se sostituite nell'equazione della retta \(r\) forniscono un'uguglianza vera: \[ 2a + 9b - 8 = 0 \qquad (*) \] Allo stesso modo il punto \(P\) appartiene alla retta \(\boldsymbol{s}\). Ovvero le coordinate \((a\,;\,\,b)\) di \(P\) se sostituite nell'equazione della retta \(s\) forniscono un'uguglianza vera: \[ -a -3b +1 = 0 \qquad (**) \] Il nostro obiettivo è trovare due numeri \(a\) e \(b\) che soddisfino le equazioni \((*)\) e \((**)\), ovvero vogliamo risolvere il sistema di equazioni \[ \begin{cases} 2a + 9b - 8 = 0 \\\\ -a -3b +1 = 0 \end{cases} \] Scegliendo un metodo qualsiasi, otteniamo la soluzione \[ \begin{cases} a = -5 \\\\ b = 2 \end{cases} \]
Soluzione: \(\,\,P\left(-5\,;\,\,2\right)\)
