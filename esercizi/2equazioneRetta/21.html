Compiti per casa


Esercizio 1

Risolvere il sistema \[ \begin{cases} 5x + 3y - 1 = 0 \\\\ -2x + 4y + 2 = 0 \end{cases} \]


Svolgimento:
La soluzione è \[ \begin{cases} x = \dfrac{5}{13} \\\\ y = -\dfrac{4}{13} \end{cases} \]




Esercizio 2

Determinare il punto di intersezionetra le rette \[ \begin{cases} -4x + 2y - 9 = 0 \\\\ -2x + 3y + 1 = 0 \end{cases} \]


Svolgimento:
La soluzione è \[ \begin{cases} x = -\dfrac{29}{8} \\\\ y = -\dfrac{11}{4} \end{cases} \]




Esercizio 3

Stabilire il punto di intersezione tra le rette \[ \begin{align*} &r:\quad -5x + 3y - 2 = 0 \\\\ &s:\quad 4x - 3y + 4 = 0 \end{align*} \]


Svolgimento:
Le rette \(r\) ed \(s\) si intersecano nel punto \(P(2\,;\,\,4)\)
