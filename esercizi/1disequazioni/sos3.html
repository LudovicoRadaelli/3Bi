Esercizio 2


Svolgere il seguente sistema di disequazioni \[ \begin{cases} 2x -4 \lt 0 \\\\ x + 1 \geq 0 \end{cases} \]


Soluzione:
Il sistema è risolto per valori della \(x\) compresi tra \(-1\) e \(2\).
In simboli: \[ -1 \leq x \lt 2 \]




Esercizio 3


Svolgere il seguente sistema di disequazioni \[ \begin{cases} -3x + 7 \geq 0 \\\\ 3x + 10 \geq 0 \end{cases} \]


Soluzione:
Il sistema è risolto per valori della \(x\) compresi tra \(-\dfrac{10}{3}\) e \(\dfrac{7}{3}\).
In simboli: \[ -\dfrac{10}{3} \leq x \leq \dfrac{7}{3} \]
