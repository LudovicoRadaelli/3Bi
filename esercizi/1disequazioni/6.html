Esercizio 1


Studiare il segno della seguente frazione al variare del valore della \(x\). \[ \dfrac{-2x + 5}{3x + 1} \]


Soluzione:
Diamo nome \(f(x)\) alla frazione, ovvero \[ f(x) := \dfrac{-2x + 5}{3x + 1} \] Si ha

\(f(x) \gt 0\,\,\) se \(\,\,-\dfrac{1}{3} \lt x \lt \dfrac{5}{2}\)

\(f(x) \lt 0\,\,\) se \(\,\, x \lt -\dfrac{1}{3}\,\,\) oppure\(\,\, x \gt \dfrac{5}{2}\)

\(f(x) = 0\,\,\) se \(x = \dfrac{5}{2}\)

\(f(x)\,\,\) non è definita se \(\,\,x = -\dfrac{1}{3}\)




Esercizio 2


Studiare il segno al variare del valore della \(x\) del prodotto di polinomi \[ (-x + 1)(-6x^2 - x + 2) \]

Soluzione:
Diamo nome \(g(x)\) al prodotto di polinomi, ovvero \[ g(x) := (-x + 1)(-6x^2 - x + 2) \] Si ha

\(g(x) \gt 0\,\,\) se \(\,\,-\dfrac{2}{3} \lt x \lt \dfrac{1}{2}\,\,\) oppure \(\,\,x \gt 0\)

\(g(x) \lt 0\,\,\) se \(\,\,x \lt -\dfrac{2}{3}\,\,\) oppure \(\dfrac{1}{2} \lt x \lt 1\)

\(g(x) = 0\,\,\) se \(\,\,x = -\dfrac{2}{3}\,\,\) oppure \(x = \dfrac{1}{2}\,\,\) oppure \(\,\, x = 1\)
