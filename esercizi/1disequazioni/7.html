Esercizio 1


Studiare il segno della seguente frazione al variare del valore di \(x\) \[ \dfrac{1 -5x}{7x - 2} \]
Soluzione:

\(\dfrac{1 -5x}{7x - 2} \gt 0\,\,\,\,\)

se \(\,\,\,\, \dfrac{1}{5} \lt x \lt \dfrac{2}{7}\)

\(\dfrac{1 -5x}{7x - 2} \lt 0\,\,\,\,\) se \(\,\,\,\, x \lt \dfrac{1}{5}\,\,\,\) oppure \(\,\,\, x \gt \dfrac{2}{7}\)

\(\dfrac{1 -5x}{7x - 2} = 0\,\,\,\,\)

se \(\,\,\,\, x = \dfrac{1}{5}\)

\(\dfrac{1 -5x}{7x - 2}\,\,\) non è definita

\(\,\,\,\)se \(\,\,\,\, x = \dfrac{2}{7}\)

Studiare il segno della seguente frazione al variare del valore di \(x\) \[ \dfrac{3x -9}{-3x +2} \]
Soluzione:

\(\dfrac{3x -1}{-3x +2} \gt 0\,\,\,\,\)

se \(\,\,\,\, \dfrac{2}{3} \lt x \lt 3\)

\(\dfrac{3x -1}{-3x +2} \lt 0\,\,\,\,\)

se \(\,\,\,\, x \lt \dfrac{2}{3}\,\,\,\) oppure \(\,\,\, x \gt 3\)

\(\dfrac{3x -1}{-3x +2} = 0\,\,\,\,\)

se \(\,\,\,\, x = 3\)

\(\dfrac{3x -1}{-3x +2}\,\,\) non è definita

\(\,\,\,\)se \(\,\,\,\, x = \dfrac{2}{3}\)




Esercizio 3


Studiare il segno della seguente frazione al variare del valore di \(x\) \[ \dfrac{x^2 -5x +6}{x -1} \]
Soluzione:

\(\dfrac{x^2 -5x +6}{x -1} \gt 0\,\,\,\,\)

se \(\,\,\,\, 1 \lt x \lt 2\,\,\,\) oppure \(\,\,\, x \gt 3\)

\(\dfrac{x^2 -5x +6}{x -1} \lt 0\,\,\,\,\)

se \(\,\,\,\, x \lt 1\,\,\,\) oppure \(\,\,\, 2 \lt x \lt 3\)

\(\dfrac{x^2 -5x +6}{x -1} = 0\,\,\,\,\)

se \(\,\,\,\, x = 2\,\,\,\) oppure \(\,\,\, x = 3\)

\(\dfrac{x^2 -5x +6}{x -1}\,\,\) non è definita

\(\,\,\,\)se \(\,\,\,\, x = 1\)
