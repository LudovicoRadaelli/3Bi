Compiti per casa


Esercizio 1

\[ \dfrac{6x^2 + 5x +1}{3x + 2} \geq 0 \]

Soluzione
\(-\dfrac{2}{3} \lt x \leq -\dfrac{1}{2}\,\,\,\) oppure \(x \geq -\dfrac{1}{3}\)




Esercizio 2

\[ (2x +1)(x^2 +3x -1)(-x) \geq 0 \]

Soluzione
\[ \begin{align*} \dfrac{-3 - \sqrt{13}}{2} \leq \,\,&x\,\, \leq -\dfrac{1}{2} \\\\ &oppure \\\\ 0 \leq \,\,&x\,\, \leq \dfrac{-3 + \sqrt{13}}{2} \end{align*} \]




Esercizio 3

\[ \begin{cases} 3x^2 - 1 \geq 0 \\\\ \dfrac{1}{3} \left(3x + 1\right) - \dfrac{x + 10}{9} \lt 0 \end{cases} \]

Soluzione
\[ x \leq -\sqrt{\dfrac{1}{3}} \,\,\,\,oppure\,\,\,\, \sqrt{\dfrac{1}{3}} \leq x \lt \dfrac{7}{8} \]
