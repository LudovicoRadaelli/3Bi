Esercizio 1


Studiare il segno della seguente frazione al variare del valore di \(x\) \[ \dfrac{-6x^2 + 11x -3}{-4x + 2} \]
Soluzione
Etichettiamo con il nome \(f(x)\) la frazione \(\dfrac{-6x^2 + 11x -3}{-4x + 2}\).

  • \(f(x) \gt 0\,\,\,\,\)

    se \(\,\,\,\, \dfrac{1}{3} \lt x \lt \dfrac{1}{2}\,\,\,\) oppure \(\,\,\,x \gt \dfrac{3}{2}\)

  • \(f(x) \lt 0\,\,\,\,\)

    se \(\,\,\,\, x \lt \dfrac{1}{3}\,\,\,\) oppure \(\dfrac{1}{2} \lt x \lt \dfrac{3}{2}\)

  • \(f(x) = 0\,\,\,\,\)

    se \(\,\,\,\, x = \dfrac{1}{3}\,\,\,\) oppure \(\,\,\,x = \dfrac{3}{2}\)

  • \(f(x)\,\) non è definita

    se \(\,\,\,\, x =\dfrac{1}{2}\)

Esercizio 2


Studiare il segno della seguente frazione al variare del valore di \(x\) \[ \dfrac{x^2 +3x + 3}{6x - 2} \]
Soluzione
  • \(\dfrac{x^2 +3x + 3}{6x - 2} \gt 0\,\,\,\,\)

    se \(\,\,\,\, x \gt \dfrac{1}{3}\)

  • \(\dfrac{x^2 +3x + 3}{6x - 2} \lt 0\,\,\,\,\)

    se \(\,\,\,\, x \lt \dfrac{1}{3}\,\,\,\)

  • \(\dfrac{x^2 +3x + 3}{6x - 2}\,\,\) non è definita

    \(\,\,\,\)se \(\,\,\,\, x = \dfrac{1}{3}\)




Esercizio 3


Studiare il segno della seguente frazione al variare del valore di \(x\) \[ \dfrac{4x^2 -11x -3}{-6x^2 - 5x + 4} \]
Soluzione
  • \(\dfrac{4x^2 -11x -3}{-6x^2 - 5x + 4} \gt 0\,\,\,\,\)

    se \(\,\,\,\, -\dfrac{4}{3} \lt x \lt -\dfrac{1}{4} \,\,\,\) oppure \(\,\,\, \dfrac{1}{2} \lt x \lt 3\)

  • \(\dfrac{4x^2 -11x -3}{-6x^2 - 5x + 4} \lt 0\,\,\,\,\)

    se \(\,\,\,\, x \lt -\dfrac{4}{3}\,\,\,\) oppure \(\,\,\, -\dfrac{1}{4} \lt x \lt \dfrac{1}{2}\)

  • \(\dfrac{4x^2 -11x -3}{-6x^2 - 5x + 4} = 0\,\,\,\,\)

    se \(\,\,\,\, x = -\dfrac{1}{4}\,\,\,\) oppure \(\,\,\, x = 3\)

  • \(\dfrac{4x^2 -11x -3}{-6x^2 - 5x + 4}\,\,\) non è definita

    \(\,\,\,\)se \(\,\,\,\, x = -\dfrac{4}{3}\,\,\,\) oppure \(x = \dfrac{1}{2}\)
